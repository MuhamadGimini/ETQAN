# كيفية تشغيل البرنامج (نسخة سطح المكتب)

هذا البرنامج هو تطبيق مكتبي مبني بتقنية Electron. لتشغيله، يجب أن يكون لديك Node.js و npm مثبتين على جهازك.

---

## الطريقة الأسهل (للمطورين)

### 1. المتطلبات الأساسية: تثبيت Node.js

*   يجب أن يكون Node.js (والذي يأتي معه npm) مثبتًا على جهازك. إذا لم يكن مثبتًا، قم بتحميله من الموقع الرسمي: **`https://nodejs.org/`**
*   نوصي بتثبيت نسخة LTS.

### 2. تثبيت الاعتماديات (Dependencies)

*   افتح موجه الأوامر (Command Prompt أو Terminal) في مجلد البرنامج.
*   قم بتشغيل الأمر التالي مرة واحدة فقط لتثبيت Electron:
    ```bash
    npm install
    ```

### 3. تشغيل البرنامج

*   بعد انتهاء التثبيت، قم بتشغيل الأمر التالي في كل مرة تريد فيها فتح البرنامج:
    ```bash
    npm start
    ```
*   سيتم فتح نافذة البرنامج تلقائيًا.

---
## للمستخدم النهائي

سيتم تزويدك بملف تثبيت `.exe` جاهز. كل ما عليك فعله هو تشغيله واتباع خطوات التثبيت. بعدها، سيظهر اختصار للبرنامج على سطح المكتب وقائمة إبدأ.
